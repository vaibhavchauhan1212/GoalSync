# 🎯 GoalSync Portal

An enterprise-grade, multi-tenant Performance Goal Setting and Lifecycle Management Portal built for modern engineering teams and agile corporate tracking cycles.

## 🚀 Live Hackathon Features
- **Role-Based Access Control (RBAC):** Distinct dashboards and security clearances for Employees, L1 Managers, and HR Administrators backed by secure JWT session authentication.
- **Real-Time Data Sync Engine:** Dynamic calculations running concurrent MongoDB Atlas telemetry collections (`Promise.all`) ensuring exact state alignment between managers and admins.
- **Technical Platform Sync Proxy:** Custom integration querying LeetCode’s GraphQL server endpoint to pull live algorithmic milestone counts straight into appraisal sheets.
- **Enterprise Rule Guardrails:** Automated token locking mechanisms and strict mathematical validation layers enforcing 100% weightage split compliance.

## 🛠️ Tech Stack
- **Frontend:** React, HTML5, CSS3, JavaScript (ES6+), Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas (Cloud Cluster Engine)
- **Security:** JSON Web Tokens (JWT), BcryptJS Cryptographic Hashing
